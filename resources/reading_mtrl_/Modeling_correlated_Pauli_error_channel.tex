\begin{abstract}
We present a Hamiltonian-free construction for temporally correlated Pauli noise suitable for stabilizer circuit simulations. 
The approach models low-frequency noise through a hidden classical Markov process that modulates Pauli error probabilities across circuit layers. 
The resulting sequence of conditional Pauli channels remains compatible with stabilizer simulators such as Stim. 
The model provides a practical approximation to slowly varying noise environments widely discussed in the quantum noise literature, including $1/f$ noise and telegraph noise processes \cite{paladino2014}.
\end{abstract}

\section{Motivation}

In realistic quantum hardware, noise processes often exhibit strong temporal correlations. 
In particular, solid-state qubit platforms such as superconducting circuits and semiconductor spin qubits are dominated by low-frequency environmental fluctuations including charge noise, flux noise, and two-level fluctuators \cite{paladino2014}. 
These processes generate error rates that drift slowly in time rather than fluctuating independently between gate operations.

Despite this, most circuit-level simulations assume independent Pauli noise after each gate layer. 
While this assumption greatly simplifies stabilizer simulation, it neglects correlations that may significantly affect logical error rates in quantum error correction. 
Several studies have shown that temporally correlated errors can degrade thresholds and lead to burst-like error events that are more difficult for decoders to correct \cite{novais2007,ball2016}.

The goal of this document is therefore to construct a temporally correlated noise model that satisfies the following requirements:

\begin{itemize}
\item Each circuit layer is affected by a Pauli channel.
\item Error rates vary slowly in time to mimic low-frequency noise.
\item The model avoids explicit Hamiltonian simulation.
\item The resulting process can be sampled efficiently and injected into stabilizer simulations such as Stim \cite{gidney2021}.
\end{itemize}

\section{Circuit model}

Consider a quantum circuit decomposed into discrete layers

\begin{equation}
C = C_N C_{N-1}\cdots C_1 ,
\end{equation}

where each $C_j$ is a Clifford gate layer.

In the standard interleaved noise picture the state evolves as

\begin{equation}
\rho_{j+1} = \mathcal{E}_j \left(C_j \rho_j C_j^\dagger\right).
\end{equation}

In the conventional i.i.d. noise model one assumes

\begin{equation}
\mathcal{E}_j = \mathcal{E}
\end{equation}

for all layers. 
However, such a model cannot capture slow drift of noise amplitudes.

To introduce correlations we instead construct a sequence of channels

\begin{equation}
\{\mathcal{E}_1,\mathcal{E}_2,\ldots,\mathcal{E}_N\}
\end{equation}

whose parameters evolve according to a stochastic process.

\section{Hidden Markov Pauli channel}

We introduce a classical hidden variable

\begin{equation}
s_j \in \{1,2,\dots,K\}
\end{equation}

representing the instantaneous noise environment.

The hidden variable evolves according to a Markov chain

\begin{equation}
P(s_{j+1}=b|s_j=a) = T_{ab},
\end{equation}

with transition matrix

\begin{equation}
T_{ab}\ge0, \qquad \sum_b T_{ab}=1.
\end{equation}

Conditioned on the environment state $s_j$, the noise channel applied at layer $j$ is a Pauli channel

\begin{equation}
\mathcal{E}^{(s_j)}(\rho)
= \sum_{P\in\mathcal{P}_n} q_{s_j}(P)\,P\rho P ,
\end{equation}

where $\mathcal{P}_n$ denotes the $n$-qubit Pauli group.

\begin{definition}
The stochastic channel sequence generated by the hidden state process $\{s_j\}$ together with the conditional Pauli distributions $\{q_a(P)\}$ is called a Hidden Markov Pauli channel.
\end{definition}

Correlated quantum channels of this form have been studied in quantum information theory as channels with memory \cite{macchiavello2002}.

\section{Two-state low-frequency noise model}

A particularly useful minimal model is obtained using two hidden states

\begin{equation}
s_j \in \{0,1\}.
\end{equation}

These states represent

\begin{itemize}
\item a low-noise environment
\item a high-noise environment
\end{itemize}

with corresponding Pauli error probabilities

\begin{equation}
p_0 = p_{\mathrm{low}}, \qquad p_1 = p_{\mathrm{high}}.
\end{equation}

The hidden state evolves according to

\begin{equation}
T =
\begin{pmatrix}
1-\gamma & \gamma \\
\gamma & 1-\gamma
\end{pmatrix}.
\end{equation}

When $\gamma$ is small the environment remains in the same state for many consecutive layers, producing strong temporal correlations. 
This discrete model closely resembles the telegraph noise processes frequently used to describe fluctuating defects in qubit environments \cite{paladino2014}.

\section{Sampling procedure}

For a circuit with $N$ layers the correlated noise process is sampled as follows.

\begin{enumerate}
\item Sample the initial environment state $s_1$ from its stationary distribution.
\item For each layer $j$:
\begin{enumerate}
\item Draw $s_{j+1}$ using the transition matrix $T$.
\item Sample a Pauli error according to distribution $q_{s_j}(P)$.
\end{enumerate}
\end{enumerate}

Because the hidden state changes slowly, consecutive layers tend to have similar error probabilities.

\section{Stim simulation workflow}

Stim is a high-performance stabilizer simulator capable of simulating large Clifford circuits with Pauli noise \cite{gidney2021}. 
However, Stim's built-in noise primitives generate independent stochastic errors.

Therefore correlated errors must be generated externally.

The recommended workflow is

\begin{enumerate}
\item Decompose the circuit into Clifford layers.
\item For each Monte Carlo shot:
\begin{enumerate}
\item Sample the hidden-state trajectory.
\item Generate the corresponding Pauli error sequence.
\item Run the circuit layer by layer in Stim.
\item Inject the sampled Pauli errors between layers.
\end{enumerate}
\end{enumerate}

This preserves stabilizer simulation efficiency while allowing arbitrary temporal correlations.

\section{Discussion}

Temporal correlations in noise have been shown to influence quantum error correction performance and decoder behavior. 
For example, correlated error models may reduce effective thresholds and produce burst errors that degrade code performance \cite{novais2007,ball2016}. 
Consequently, incorporating such correlations into circuit-level simulations is important for realistic benchmarking of quantum architectures.

The Hidden Markov Pauli channel provides a flexible and computationally efficient framework for incorporating these effects while remaining compatible with stabilizer-based simulators.

\section{Conclusion}

We constructed a Hamiltonian-free model for temporally correlated Pauli noise based on a hidden Markov process. 
The model captures low-frequency drift in error rates while remaining compatible with stabilizer circuit simulation tools such as Stim.
